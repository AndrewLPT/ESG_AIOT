This web application provides the following features:

  • Interactive sliders and input fields for adjusting the parameters (slope, intercept, noise factor, and number of points).
  • A scatter plot of the generated data points (in blue) and the regression line (in red).
  • Real-time updates of the chart and results as you adjust the parameters.
  • Display of the estimated slope, intercept, and R-squared score.

The application uses:

  • Chart.js for plotting the scatter plot and regression line.
  • regression-js for performing the linear regression calculation.
  • Custom JavaScript to generate the data points and handle user interactions.

This web-based version allows users to interact with the Linear Regression Solver directly in their browser without the need for Python or any additional installations. It provides a similar experience to the Streamlit app but runs entirely in the browser.
